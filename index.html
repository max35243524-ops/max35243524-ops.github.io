Landis+Gyr — Test Fixture UI

PyQt6 app that interfaces with serial instruments and CT hardware for automated testing (SCPI, I²C, GPIO). Includes jog sequencing and safe source activation.

  • Manual control and scripted runs for current/voltage sources (SCPI over serial).
  • Non‑blocking waits; strict sequencing: force‑off → CT setup → routing → safe activation.
  • Logs, progress, and error handling for long‑running tests.

Inline Code Runner (Client‑side)

Run small code snippets directly in the browser. Python runs via Pyodide (WebAssembly); JavaScript runs natively. Great for demos, not heavy packages.

Limits: browser‑only (no internet access), Python wheels are restricted to what Pyodide includes. For heavier demos, link a Colab or embed a Replit/StackBlitz.

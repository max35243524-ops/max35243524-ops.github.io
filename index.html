Work at Landis+Gyr

Details about my contributions at Landis+Gyr.

The video on the left shows a UI that I was developing that interfaces with a electrical meter testing fixture. I had to interface with an Arduino that translated commands from the UI to SCPI commands that are sent to the voltage and current sources stacked below the meters. The UI was originally written in C and I was tasked with translating it into python. The video on the right features a current transformer board that I took part in the design of, and soley wrote the code of to control the balast rresistors for varbiale control of the output current when the board is attached to the three CTs, going to a maximum of 600 amps.
